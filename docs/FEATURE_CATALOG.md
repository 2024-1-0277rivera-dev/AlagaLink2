
# AlagaLink: Feature Catalog

## 1. Universal Search (Command Palette)
- **Location**: Navbar (Desktop & Mobile).
- **Capabilities**: Cross-entity search for Pages, Portals, Members, Inventory, and News.

## 2. Member Management (Admin)
- **Registry**: Filtered table for Active, Pending, and Suspended accounts.
- **Unified Profile Editing**: Ability to update user credentials using the structural sections of the Registration Form.
- **Registration**: Multi-section comprehensive form including account linking (Gmail/Facebook).
- **IDEA Categories**: Integrated picker for the 14 IDEA disability categories.

## 3. Programs & Services
- **PWD ID Portal**: Search existing members, browse registry popover, or register new.
- **Inventory Portals**: Assistive Devices, Medical Assistance, PhilHealth, and Livelihood.

## 4. News & Engagement
- **Dynamic Hero Carousel**: Auto-rotating slideshow of municipal updates with advanced animations.

## 5. Lost & Found Portal
- **Reporting**: Restricted to Admin/PDAO Staff to ensure high-integrity reporting and registry verification.
- **Resolution**: Narrative logging for found cases (What/When/Why/How).

## 6. Assistive Tools
- **AIagaLink Chatbot**: Gemini-powered assistant for system guidance.
- **Notification Drawer**: Real-time alerts for ID readiness and system updates.
